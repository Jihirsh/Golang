Simple Calculator
📌 Problem

Create a command-line calculator in Go that:

Takes two integer inputs from the user.
Takes an arithmetic operator (+, -, *, /).
Performs the selected operation.
Prints the result.
Handles invalid number input.
Prevents division by zero.
Displays an error for unsupported operators.


📥 Example Input
Enter first number:
15

Enter operator (+ - * /):
*

Enter second number:
8
📤 Example Output
120



🧠 Approach
Read the first integer.
Read the operator.
Read the second integer.
Use a switch statement to determine which operation to perform.
Call a separate function for each mathematical operation.
For division:
Check whether the divisor is zero.
Return an error if division by zero is attempted.
Print the result.


✨ Features
User input using fmt.Scan()
Separate function for each operation
Uses switch statements
Proper error handling
Division by zero protection
Clean and modular code
